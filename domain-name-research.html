Domain Name Research

Questions:

What is an internet domain name?

An internet domain name is an address which people can use in order to access different sites on the internet.

Why are domain names so important on the web?

Not only do the web names represent the site they are associate with, but the also make it immensely easier to navigate the web.

Why do domain names have to be unique?

Each domain name is attached to a unique IP address in order to access different sections of the web.

What does it mean to register a domain name?

To register a domain name is to gain ownership of it and attach code to it so it can be accessed by other internet users.

If you wanted a domain name that is already taken, how could you get it?

You can purchase that name from the owner of the domain name.
